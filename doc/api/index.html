dart_libp2p_kad_dht

Dart libp2p Kademlia DHT

A comprehensive Dart implementation of the libp2p Kademlia Distributed Hash Table (DHT) for building decentralized peer-to-peer applications. This library provides the core infrastructure for peer discovery, content routing, and distributed key-value storage in P2P networks.

üåü Features

  • Peer Discovery: Find peers by their ID across the network
  • Content Routing: Discover who has specific content using content addressing
  • Distributed Storage: Store and retrieve key-value pairs across the network
  • Service Discovery: Advertise and find services in the P2P network
  • Network Modes: Support for client, server, and auto modes
  • Bootstrap Integration: Easy connection to existing libp2p networks
  • Mobile Optimized: Resource-efficient operation for mobile devices
  • High Performance: Optimized routing tables and query algorithms
  • Comprehensive Testing: Extensive test suite with real network scenarios

üöÄ Quick Start

Installation

Add the package to your pubspec.yaml:

dependencies:
  dart_libp2p_kad_dht: ^1.1.0
  dart_libp2p: ^0.5.2
  dcid: ^1.0.0

Basic Usage

import 'package:dart_libp2p_kad_dht/dart_libp2p_kad_dht.dart';
import 'package:dart_libp2p/dart_libp2p.dart';

Future<void> main() async {
  // Create a libp2p host
  final host = await createLibp2pHost();
  
  // Create a provider store for content routing
  final providerStore = MemoryProviderStore();
  
  // Create and start the DHT
  final dht = IpfsDHT(
    host: host,
    providerStore: providerStore,
    options: const DHTOptions(
      mode: DHTMode.auto,
      bucketSize: 20,
      concurrency: 10,
    ),
  );
  
  await dht.start();
  await dht.bootstrap(); // Connect to the network
  
  // Find a peer
  final peerInfo = await dht.findPeer(targetPeerId);
  
  // Store a value
  await dht.putValue('my-key', utf8.encode('my-value'));
  
  // Retrieve a value
  final value = await dht.getValue('my-key');
  
  // Cleanup
  await dht.close();
  await host.close();
}

üìö Examples

Interactive P2P Node

Run a fully interactive P2P node with all DHT operations:

dart run examples/basic_p2p_node.dart

Available Commands:

  • stats - Show network statistics
  • store <key> <value> - Store a key-value pair
  • get <key> - Retrieve a value
  • announce <content-id> - Announce content
  • find-content <content-id> - Find content providers
  • find-peer <peer-id> - Find a specific peer
  • quit - Exit the demo

Mobile Optimized Node

For resource-constrained environments:

dart run examples/mobile_p2p_node.dart

Server Node

High-performance server deployment:

dart run examples/server_node.dart --port 4001

üîß Configuration

DHT Options

final dhtOptions = DHTOptions(
  mode: DHTMode.auto,        // client, server, or auto
  bucketSize: 20,            // K-bucket size
  concurrency: 10,           // Concurrent operations
  resiliency: 3,             // Query redundancy
  bootstrapPeers: [          // Network entry points
    AddrInfo(peerId1, [addr1]),
    AddrInfo(peerId2, [addr2]),
  ],
);

Network Modes

  • Client Mode: Lightweight, doesn't store routing data
  • Server Mode: Full routing table, serves other peers
  • Auto Mode: Automatically switches based on network conditions
