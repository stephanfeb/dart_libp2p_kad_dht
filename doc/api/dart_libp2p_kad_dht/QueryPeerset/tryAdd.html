Tries to add the peer p to the peer set. If the peer is already present, no action is taken. Otherwise, the peer is added with state set to PeerState.heard. Returns true if the peer was not already present.

Implementation

bool tryAdd(PeerId p, PeerId referredBy) {
  if (_find(p) >= 0) {
    return false;
  } else {
    _all.add(_QueryPeerState(
      id: p,
      distance: _distanceToKey(p),
      state: PeerState.heard,
      referredBy: referredBy,
    ));
    _sorted = false;
    return true;
  }
}
