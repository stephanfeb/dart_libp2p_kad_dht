QueryPeerset

QueryPeerset class

QueryPeerset maintains the state of a Kademlia asynchronous lookup. The lookup state is a set of peers, each labeled with a peer state.

Constructors

QueryPeerset.new(List<int> key)
Creates a new empty set of peers. key is the target key of the lookup that this peer set is for.
