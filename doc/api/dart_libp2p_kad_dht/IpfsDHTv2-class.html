IpfsDHTv2 class

IpfsDHT v2 - A modular, maintainable implementation of the IPFS DHT

This implementation provides the same public API as the original IpfsDHT but with improved architecture, better error handling, and enhanced observability.

Key improvements:

  • Modular design with focused components
  • Consistent query patterns with unified error handling
  • Built-in metrics and monitoring
  • Simplified configuration
  • Better testability
